## shii-bot 오픈소스 입니다.

/say: 저장된 단어를 출력                            
/copy: 메세지 복제                            
/game, mining, roll: 미니게임                            
/clear: 메세지 청소                            
/start, stop: 음성채널입장,퇴장(음성채널기능은 추후 업데이트 예정)                            
/출첵, 누적출석수: 출석체크 및 누적출석수 확인                            
/embed: shii-bot 프로필                            
/hello, bye: 인사
